# Road Safety Analytics — Tableau Project

A Business Intelligence study focused on road accident mortality and the effectiveness of safety equipment. This project uses **Tableau** to visualize data trends across the European Union and Greece.

## Key Features

* **Data Cleansing:** Processed raw data from ELSTAT, EUROSTAT, and the WHO using **Tableau Prep**.

* **EU Mortality Trends:** Map and area charts showing death rates per 100,000 people across all EU member states.

* **Helmet Efficacy:** Treemaps and bubble charts demonstrating the correlation between helmet usage and survival rates for two-wheeled vehicles.

* **Regional Analysis:** A **Hex Map of Greece** that allows users to toggle views between Municipalities and Regional Units.
